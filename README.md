# Academic Programming & Cybersecurity Foundation Portfolio

Welcome to my practical engineering portfolio. This repository contains core programming implementations from my 12th Standard State Board curriculum alongside ongoing practical foundations in technical computer systems.

---

## 📁 Repository Contents

### 1. Object-Oriented Sphere Calculator (`area_of_sphere.cpp`)
* **Concepts Demonstrated:** Object-Oriented Programming (OOP), Class Structure, Data Encapsulation (`public`/`private`), and Objects.
* **Description:** A modular program that models a physical sphere as a class object, safely protecting internal data variables while exposing public functions to process and compute surface area.

### 2. Algorithmic Max-Value Finder (`largest_of_four.cpp`)
* **Concepts Demonstrated:** Procedural Programming, Function Prototypes, Parameter Passing, and Sequential Selection Logic.
* **Description:** A script based on Class 12 board examination patterns that inputs four distinct integers and utilizes a sequential elimination scan within a user-defined function to determine the maximum value efficiently.
